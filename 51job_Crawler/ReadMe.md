# 利用BeautifulSoup+requests+xlwt进行51Job的职位爬取  

## 相关配置

**Installation**  

Do the following in your virtualenv:  

*pip install xlwt*

|FileName|Notes|  
|-------|----------|  
|51job_bs_1.py|首次提交代码|  
|51job_bs_2.py|修复爬取工作简介内容不连续|  
|51Job_bs_C++.xls|c++职位信息表|
|51Job_bs_python.xls|python职位信息表|
